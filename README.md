# JS Calculator

This project is a simple calculator for basic mathematical operations (addition, subtraction, multiplication, division) built using HTML, CSS, and JavaScript.

## MVP
This calculator implements the following functions:
- Addition (+)
- Subtraction (−)
- Multiplication (×)
- Division (÷)
- Clear (C) button
- Backspace (⌫) button
- Toggle sign (±)
- Percentage (%)

## Technologies
- HTML — Structure of the calculator
- CSS — Styling the interface
- JavaScript — Calculator logic

## How to Run
1. Clone the repository or download the files to your computer.
2. Open the index.html file in your browser.
3. The calculator will work without any additional setup.

## How to Use
- Enter numbers and operations using the on-screen buttons or your keyboard.
- To complete a calculation, press the "=" button or hit the Enter key.
- To clear the screen, press the "C" button.
- To delete the last entered symbol, use the "⌫" button.

## Example Usage
Enter:
- 7 + 3 = to get the result 10
- 8 × 5 = to get the result 40
- 9 ÷ 3 = to get the result 3
